Fill

Este es el valor predeterminado que estira la imagen para que se ajuste al cuadro de contenido,independientemente de su relación aspecto.

Contain

aumenta o disminuye el tamaño de la imagen vertical u horizontalmente para llenar el cuadrado conservando su relación de aspecto

Cover

se adapta al tamaño de pantall rellenando

None

La imagen ignorara la altura y ancho del padre y conservara su tamaño

Scale down

Lla imagen comparara el tamaño entre el valor none y el containy mostrar el que tenga el valor mas pequeño
